# Evaluación T2 - Cliente Web Biblioteca

## Autor
**Nombre:** Jamir Ascencio Flores

## Descripción
Aplicación web cliente desarrollada en React + Vite para consumir la API de Biblioteca.
Permite gestionar libros (CRUD) y registrar préstamos validando el stock disponible.

## Tecnologías
* React 18 (Vite)
* Bootstrap 5
* Axios
* React Router DOM

## Instrucciones de Ejecución

1. **Instalar dependencias**:
   ```bash
   npm install
   ```
2. **Configurar API**: Verificar que la URL del backend en `src/services/api.js` coincida con el puerto de su API local.

3. **Ejecutar proyecto**:
   ```bash
    npm run dev
   ```
